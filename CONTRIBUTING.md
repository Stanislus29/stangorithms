# Contributor Guidelines

We welcome contributions to **StanLogic** to advance open computational education and research in Boolean logic and digital systems.

By submitting a pull request or issue, you agree that:
- Your contributions are provided under the same **MIT License** as this repository.
- You grant the maintainers the right to use, modify, and distribute your contributions as part of the project.
- Attribution will be preserved through commit history and acknowledgments.

### How to Contribute
1. Fork the repository and create a new branch for your feature or fix.
2. Write clear, well-documented code following Python best practices.
3. Add or update tests and documentation if relevant.
4. Submit a pull request describing your changes.

Contributors are acknowledged in release notes, research references, or future publications when applicable.
